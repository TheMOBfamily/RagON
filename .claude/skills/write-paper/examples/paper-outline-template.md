## Section Outline


### 1. Introduction (~10%)

**Goal:** Hook reader, establish gap, state contribution.

1.1 Context
- {Broad importance of topic}
- {Recent developments}

1.2 Gap
- {What's missing in literature}
- {Why it matters}

1.3 Contribution
- {This paper does X}
- {We find that Y}

1.4 Structure
- {Paper organization}


### 2. Literature Review (~15%)

**Goal:** Position paper in existing knowledge.

2.1 Theme A
- {Key findings from literature}
- {Gap identified}

2.2 Theme B
- {Key findings}
- {Contradiction/gap}

2.3 Theoretical Framework
- {Framework used}
- {Justification}


### 3. Methodology (~15%)

**Goal:** Enable replication.

3.1 Research Design
- {Type: quantitative/qualitative/mixed}

3.2 Data
- {Source, period, sample size}
- {Selection criteria}

3.3 Variables
- {Dependent variable(s)}
- {Independent variable(s)}
- {Control variable(s)}

3.4 Analysis
- {Statistical methods}
- {Software and version}


### 4. Results (~25%)

**Goal:** Present findings objectively.

4.1 Descriptive Statistics
- Table 1: Summary statistics
- Figure 1: Distribution

4.2 Main Results
- Table 2: Main analysis
- Finding 1: {Primary result with numbers}
- Finding 2: {Secondary result}

4.3 Robustness Checks
- {Alternative specifications}
- {Sensitivity analysis}


### 5. Discussion (~20%)

**Goal:** Interpret findings, compare with literature.

5.1 Interpretation
- {What do results mean?}
- {Why did we find this?}

5.2 Comparison
- {How does it fit literature?}
- {Agreements/disagreements}

5.3 Implications
- {Theoretical implications}
- {Practical implications}

5.4 Limitations
- {Honest assessment}
- {How they affect conclusions}


### 6. Conclusion (~5%)

**Goal:** Summarize, highlight contribution.

6.1 Summary
- {2-3 sentence recap}

6.2 Contribution
- {Main takeaway}

6.3 Future Research
- {Next steps}
