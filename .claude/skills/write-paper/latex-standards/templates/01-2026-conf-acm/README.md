# ACM LaTeX Template (acmart)

**Source:** https://github.com/borisveytsman/acmart
**Version:** 2.16 (August 2025)

## Usage

```latex
\documentclass[sigconf]{acmart}  % For conference proceedings
\documentclass[acmsmall]{acmart} % For most journals
```

## Files

- `acmart-primary/` - Full template package
- `acmart.cls` - Main document class
- `ACM-Reference-Format.bst` - Bibliography style

## Notes

- Font: Libertine (required, cannot substitute)
- Margins: Fixed (cannot adjust)
- Metadata: Plain ASCII only (no diacritics in title/author)
