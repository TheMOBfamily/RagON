# ICML 2024 LaTeX Template

**Source:** https://icml.cc/Conferences/2024/AuthorInstructions
**Version:** 2024

## Usage

```latex
\usepackage{icml2024}
```

## Files

- `icml2024/` - Full template package
- `icml2024.sty` - Style file
- `example_paper.tex` - Template
- `icml2024-example.pdf` - Example output

## Format Requirements

- Two-column format
- Standard ML conference layout
- Similar to NeurIPS formatting

## Notes

- Check papercheck.icml.cc before submission
- No Type 3 fonts
- Vector graphics preferred (EPS/PDF)

[ICML Author Instructions](https://icml.cc)
