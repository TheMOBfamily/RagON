# NeurIPS 2023 LaTeX Template

**Source:** https://neurips.cc/Conferences/2023/PaperInformation/StyleFiles
**Version:** 2023

## Usage

```latex
\usepackage{neurips_2023}           % For submission
\usepackage[final]{neurips_2023}    % For camera-ready
\usepackage[preprint]{neurips_2023} % For arXiv preprint
```

## Files

- `neurips_2023.sty` - Style file
- `neurips_2023.tex` - Template with instructions
- `neurips_2023.pdf` - Example output

## Format Requirements

- Text area: 5.5" x 9"
- Left margin: 1.5"
- Font: 10pt, Times New Roman preferred
- Leading: 11 points
- Page limit: 9 content pages + unlimited references

## Notes

- Checklist entered in OpenReview (not in PDF)
- Max file size: 50MB
- Supplementary: up to 100MB (PDF/ZIP)
