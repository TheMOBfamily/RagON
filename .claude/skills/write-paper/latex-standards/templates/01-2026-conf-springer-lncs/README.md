# Springer LNCS LaTeX Template

**Source:** CTAN - https://ctan.org/pkg/llncs
**Use case:** Lecture Notes in Computer Science proceedings

## Usage

```latex
\documentclass{llncs}
```

## Files

- `llncs/` - Full template package
- `llncs.cls` - Main document class
- `splncs04.bst` - Bibliography style

## Format Requirements

- Single column layout
- Numbered sections
- Author-year or numerical citations supported

## Notes

- Content-first approach
- No custom fonts
- Convert special chars to TeX codes (ä → \"a)
