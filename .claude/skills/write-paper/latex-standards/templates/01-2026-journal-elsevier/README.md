# Elsevier LaTeX Template (elsarticle)

**Type:** Journal
**Source:** CTAN - https://ctan.org/pkg/elsarticle
**Version:** December 2024

## Usage

```latex
\documentclass[preprint,12pt]{elsarticle}   % Preprint
\documentclass[review]{elsarticle}          % Review (double-spaced)
\documentclass[final,5p,times]{elsarticle}  % Final (two-column)
```

## Files

- `elsarticle/` - Full template package
- `elsarticle.cls` - Document class
- `elsarticle-num.bst` - Numerical bibliography style
- `elsarticle-harv.bst` - Harvard (author-year) style

## PDF Examples

- `doc/elsdoc.pdf` - User documentation
- `doc/elstest-1p.pdf` - Single column example
- `doc/elstest-3p.pdf` - Three-column example
- `doc/elstest-5p.pdf` - Five-column example

## Format Requirements

- Covers ~2500+ Elsevier journals
- Including: Computers & Education, Internet and Higher Education
- Single or two-column layout depending on journal
