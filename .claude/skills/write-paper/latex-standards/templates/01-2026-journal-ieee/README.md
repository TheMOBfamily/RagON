# IEEE LaTeX Template (IEEEtran)

**Source:** CTAN - https://ctan.org/pkg/ieeetran
**Version:** 1.8b

## Usage

```latex
\documentclass[conference]{IEEEtran}
```

## Files

- `IEEEtran/` - Full template package
- `IEEEtran.cls` - Main document class
- `IEEEtran.bst` - Bibliography style

## Format Requirements

- Paper: US Letter (8.5" x 11") or A4
- Layout: Two columns
- Font: Times New Roman, 10pt body
- Title: 24pt, Author: 11pt

## Notes

- No Type 3 fonts allowed
- Title capitalization rules apply
