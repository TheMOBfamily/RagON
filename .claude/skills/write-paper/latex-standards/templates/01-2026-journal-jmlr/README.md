# JMLR LaTeX Template

**Source:** TeX Live system (texlive-publishers)
**Use case:** Journal of Machine Learning Research

## Usage

```latex
\documentclass{jmlr}
```

## Files

- `jmlr.cls` - Main document class for articles
- `jmlrbook.cls` - Document class for books/proceedings
- `jmlrutils.sty` - Utility macros

## Notes

- Standard for ML journal publications
- Clean, academic formatting
- Compatible with most ML conferences
