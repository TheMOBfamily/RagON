# MDPI LaTeX Template

**Type:** Journal (Open Access)
**Source:** https://www.mdpi.com/authors/latex
**Version:** January 2026 (APA citation style)

## Usage

```latex
\documentclass[journal,article]{mdpi}
```

## Files

- `mdpi/` - Extracted template
- `mdpi/template.tex` - Main template file
- `mdpi/template.pdf` - Example output
- `mdpi/Definitions/` - Class file, logos, bibliography styles

## PDF Examples

- `mdpi/template.pdf` - Compiled example

## Format Requirements

- Open access journals (Education Sciences, Sustainability, etc.)
- Single column layout
- APA, ACS, or Chicago citation styles available

## Notes

- Template variants: APA, ACS, Chicago (this is APA)
- Overleaf template also available
