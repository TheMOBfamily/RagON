# SAGE LaTeX Template

**Type:** Journal
**Source:** https://uk.sagepub.com/en-gb/eur/manuscript-submission-guidelines
**Version:** 2020

## Usage

```latex
\documentclass[Afour,times,sageh]{sagej}  % A4, Times font, Harvard style
\documentclass[Afour,times,sagev]{sagej}  % A4, Times font, Vancouver style
```

## Files

- `sage/` - Extracted template
- `sage/sagej.cls` - Document class
- `sage/SageH.bst` - Harvard bibliography style
- `sage/SageV.bst` - Vancouver bibliography style

## PDF Examples

- `sage/Sage_LaTeX_Guidelines.pdf` - LaTeX guidelines

## Format Requirements

- Covers SAGE journals (e.g., J. Educational & Behavioral Statistics)
- Harvard or Vancouver citation styles
- Single column layout
