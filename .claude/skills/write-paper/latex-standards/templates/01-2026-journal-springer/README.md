# Springer Nature Journal LaTeX Template (sn-jnl)

**Type:** Journal
**Source:** https://www.springernature.com/gp/authors/campaigns/latex-author-support
**Version:** 3.1 December 2024

## Usage

```latex
\documentclass[pdflatex,sn-basic]{sn-jnl}       % Basic reference style
\documentclass[pdflatex,sn-mathphys-num]{sn-jnl} % Math/Physics numbered
\documentclass[pdflatex,sn-apa]{sn-jnl}          % APA style
\documentclass[pdflatex,sn-chicago]{sn-jnl}      % Chicago humanities
\documentclass[referee,sn-basic]{sn-jnl}          % Double-spaced review
```

## Files

- `sn-jnl/sn-article-template/` - Full template
- `sn-jnl.cls` - Document class
- `sn-article.tex` - Template file
- `bst/` - Bibliography style files

## PDF Examples

- `sn-jnl/sn-article-template/sn-article.pdf` - Example output
- `sn-jnl/sn-article-template/user-manual.pdf` - User manual

## Format Requirements

- Content-first approach, minimal formatting
- Covers Springer Nature, Nature Portfolio, BMC journals
- Single column layout

## Notes

- Khác với Springer LNCS (conference) - đây là template cho journals
