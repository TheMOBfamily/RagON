# Paper Structure - IMRaD Guide

> Cau truc chuan cho bai bao khoa hoc.


## IMRaD Structure

| Section | Purpose | Length |
|---------|---------|--------|
| **I**ntroduction | Why? Gap, contribution | 10-15% |
| **M**ethods | How? Replicable steps | 15-20% |
| **R**esults | What? Findings with data | 25-30% |
| **a**nd | - | - |
| **D**iscussion | So what? Interpretation | 25-30% |


## Section Details


### Introduction

**Formula:** Gap → Contribution → Structure

| Element | Content |
|---------|---------|
| Hook | Broad context, importance |
| Gap | What's missing in literature |
| Objective | Research question/hypothesis |
| Contribution | What this paper adds |
| Structure | Paper organization |


### Methods

**Formula:** What + How + Why

| Element | Content |
|---------|---------|
| Study design | Type of research |
| Sample | N, selection criteria |
| Data collection | Instruments, procedures |
| Analysis | Statistical methods |
| Ethics | IRB approval if needed |

**Rule:** Enough detail for replication.


### Results

**Formula:** Answer → Evidence → Pattern

| Element | Content |
|---------|---------|
| Main finding | Primary result first |
| Supporting data | Tables, figures |
| Secondary findings | Additional results |
| Unexpected results | Anomalies |

**Rule:** No interpretation here. Just facts.


### Discussion

**Formula:** Meaning → Comparison → Limitation → Implication

| Element | Content |
|---------|---------|
| Key findings | Restate main results |
| Interpretation | What does it mean? |
| Comparison | How does it fit literature? |
| Limitations | Honest assessment |
| Implications | Practical/theoretical |
| Future work | Next steps |


### Conclusion

**Formula:** Summary → Contribution → Future

| Element | Content |
|---------|---------|
| Summary | Brief recap (2-3 sentences) |
| Contribution | Main takeaway |
| Future | Next directions |

**Rule:** No new information. No repeat of results.


### Abstract

**Formula:** Background → Objective → Methods → Results → Conclusion

| Element | Words |
|---------|-------|
| Background | 1-2 sentences |
| Objective | 1 sentence |
| Methods | 1-2 sentences |
| Results | 2-3 sentences |
| Conclusion | 1-2 sentences |
| **Total** | 150-300 words |

**Rule:** Write LAST. After paper complete.


---


## Writing Order (IMPORTANT)

```
1. Methods     - What you did
2. Results     - What you found
3. Discussion  - What it means
4. Introduction - Why it matters
5. Conclusion  - Summary
6. Abstract    - LAST
```


## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Abstract first | Write LAST |
| Methods vague | Add details for replication |
| Results + Discussion mixed | Separate clearly |
| Introduction too long | Focus on gap |
| No limitations | Be honest |
| Conclusion repeats | Be brief, no new info |
