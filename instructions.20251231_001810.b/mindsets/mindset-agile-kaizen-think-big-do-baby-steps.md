# Agile-Kaizen: Think Big, Do Baby Steps

> Build small → Ship fast → Verify → Repeat

## Mantras

1. Think big. Act small.
2. Build small. Ship fast.
3. Fail fast. Fix fast.
4. Commit often. Push often.
5. Query while coding.
6. Progress over perfection.

## Checklist

- [ ] Vision clear?
- [ ] Task ≤2h? (Split if larger)
- [ ] Deliverable working?
- [ ] Committed at milestone?
- [ ] Pushed before switch?

## Patterns

```
Work → Milestone → Commit → Push → Repeat
Query → Act → Query → Act → Done
```

## {git flow} (MANDATORY)

```bash
git add . && git commit -m "message" && git push origin
```

| ❌ Wrong | ✅ Correct |
|----------|-----------|
| 2h → 1 commit | 15-30 min → commit |
| "Để lát commit" | Commit ngay |

**Commit frequency**: Mỗi 15-30 min. Mỗi function. Mỗi fix.

## Agile File Creation

```
❌ BATCH: Do all → Write at end
✅ INCREMENTAL: Skeleton → Fill → Commit → Repeat
```

## Task Size

- ≤2h → Do it
- 1-2 days → Split
- 1+ week → Decompose

**One-liner**: Think big. Act small. Ship fast. Verify. Repeat.
