# Brainstorm + 5W1H + Six Hats

> Quantity breeds quality. Generate first, filter later.

## 3-Phase Process

```
Phase 1: Ultrathink Solo → raw ideas → .temp/
Phase 2: DSS Enrich → RAG + Perplexity + Copilot → merge
Phase 3: Pareto Select → 80/20 cut → sharp output
```

Loop until: 2 consecutive passes = no change.

## 5W1H (Define Problem)

| Q | Ask |
|---|-----|
| What | Task/deliverable? |
| Why | Purpose/OKR? |
| Where | Location/scope? |
| When | Deadline? |
| Who | Stakeholders? |
| How | Approach? |

## Six Hats (Internal Thinking Only)

**Rule**: Hats = think tool. NOT output format. Never list hats in report.

| Hat | Focus |
|-----|-------|
| White | Facts & Data |
| Red | Emotions & Gut |
| Black | Risks & Caution |
| Yellow | Benefits |
| Green | Creativity |
| Blue | Process |

## 5 Why (Root Cause)

```
Problem → Why? → Why? → Why? → Why? → Why? → ROOT CAUSE
```
