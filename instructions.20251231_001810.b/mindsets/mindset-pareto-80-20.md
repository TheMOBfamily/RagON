# Pareto 80/20 Mindset

> 20% effort → 80% results. Focus on vital few.

## Core Principle

80% outcomes come from 20% causes.
Identify vital few. Ignore trivial many.

## When to Apply

- Task list overwhelming → Pick top 20%.
- Too many rules → Focus critical few.
- Limited time → Maximum impact actions.
- Analysis paralysis → Decide on essentials.

## How to Apply

1. List ALL items (30+).
2. Rate impact (★ to ★★★★★).
3. Calculate 20%: `total * 0.2`.
4. Select top N by impact.
5. Execute top N FIRST.
6. Others: do later or skip.

## Formula

```
Top N = Total × 0.2
Example: 30 rules × 0.2 = 6 rules
```

## Anti-Patterns

- Trying to do everything.
- Equal time on all tasks.
- Perfectionism on low-impact items.
- Analysis without prioritization.

## Examples

### Writing Guide
- 30 grammar rules → 6 critical rules
- Focus: sentence length, clarity, consistency

### Code Review
- 50 issues → 10 blockers
- Focus: security, correctness, performance

### Learning
- 100 topics → 20 fundamentals
- Focus: core concepts, daily use cases
