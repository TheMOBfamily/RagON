# Step-by-Step Exact Execution

> Reading ≠ Doing. Execute EXACTLY. Skip NOTHING.

## Core Principle

H₀: "I will skip steps" — Prove otherwise by DOING each step.

## 3 Principles

| # | Principle | Action |
|---|-----------|--------|
| 1 | Execute Exactly | Every step. No skip. No shortcut. |
| 2 | Show Work | Prove each step. Visible output. |
| 3 | Tick as Done | Mark ONLY after actual execution. |

## Anti-Skip Protocol

```
BEFORE: Read entire checklist
DURING: Execute ONE step at a time
AFTER:  Mark done with evidence
```

## Rules

1. One step = One action. Complete before next.
2. Tick = Evidence. "Done" requires proof.
3. Skip = Failure. Any skip = task incomplete.
4. Assume nothing. Each step mandatory.

## Why AI Skips

1. Cognitive shortcuts
2. Overconfidence ("I know this")
3. Pattern matching
4. Context window limits
5. Confirmation bias

## Prevention Mantras

- Đọc không phải là làm.
- Mỗi bước phải có bằng chứng.
- Bỏ qua = thất bại.
